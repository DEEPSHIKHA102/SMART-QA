# SmartQA – Real-Time AI-Powered Q&A Platform

SmartQA is a real-time question-and-answer platform tailored for events like lectures, workshops, or webinars. It allows event hosts to create secure rooms and enables participants to submit questions live during the session.

## Features
- Real-time question posting and updates using Socket.IO
- Gemini AI API integration to rank and filter top questions
- JWT-secured authentication with role-based access (host/participant)
- Decoupled frontend-backend (microservice architecture)
- MongoDB for dynamic session and user data handling

## Tech Stack
- **Frontend**: React.js
- **Backend**: Node.js, Express.js
- **Real-time**: Socket.IO
- **AI API**: Gemini AI API
- **Database**: MongoDB
- **Auth**: JWT, Role-Based Access Control
